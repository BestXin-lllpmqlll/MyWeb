# 杀戮尖塔(Slay the Spire)风格卡牌转场设计

## 1. 设计概述 (Overview)
这是一个从首页（`/`）跳转至详情页（`/main`）的互动式过渡层（`/selection`）。旨在用经典卡牌游戏“选牌”的体验，增加浏览作品集前的趣味性和沉浸感。

## 2. 交互流程 (User Flow)
1. **暗幕揭开**：点击首页 `ENTER` 波纹展开后，进入纯黑色的 `/selection` 页面。
2. **卡牌入场**：三张卡牌（左、中、右）以稍微交错的时间差，从屏幕下方平滑上浮并带有淡入效果，最终形成一个扇形（Fan-out）的展开结构（左右略微倾斜）。
3. **悬浮交互 (Hover)**：
   - 鼠标悬停在卡牌上时，卡牌会回正（旋转归零）、略微放大，并伴有内发光/外发光的反馈。
   - 只有中间的 `PORTFOLIO` 卡牌是可点击的，左右两张设为“锁定”或“未解锁”状态（点击无效且呈现警告边框）。
4. **选中动画 (Select)**：
   - 点击中间的卡牌后，其余两张卡牌瞬间淡出消失。
   - 选中的卡牌放大至屏幕中央，伴随一阵闪白光效（Flash），仿佛卡牌力量被激活。
   - 延迟 `1.2s` 后，顺畅跳转至详情页 `/main`。

## 3. 技术实现 (Technical Implementation)
- **状态管理**：使用 `selectedCard` (number | null) 控制动画的分支状态。
- **扇形排列**：利用绝对定位结合内联 `style` 的 `transform: rotate(x) translateX(y) translateY(z)` 实现卡牌的初始扇面排布。
- **自定义动画**：
  - `card-enter`：控制 Y 轴平移与透明度，实现错落有致的入场效果。
  - `shimmer`：利用渐变背景和 `skew` 实现卡牌表面的扫光特效。
  - `flash`：选中时全卡闪白的冲击感。
